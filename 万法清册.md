## 八 · 狀態契約 (Res)

所有 `道.*` 方法 (除 `活体.connect()` 返 Dao 對象以供鏈式調用外) 返 `Res`:

```python
Res = {
    "ok":        bool,         # 成功與否
    "data":      Any,          # 主體結果
    "warnings":  List[str],    # 非致命警告
    "error":     Optional[str], # 致命錯誤 (ok=False 時)
    "elapsed_s": float,         # 用時秒數 (若量測)
}
```

Agent 代碼範式:

```python
r = 道.反.外("..."); data = r.data if r.ok else None
r = 道.秀.load("p.step")
if not r.ok: print(r.error); return
```

`Res` 同時支持 dict 訪問 (`r["ok"]`) 與 attr 訪問 (`r.ok`), 無歧義.

## 九 · 三原則 (agent 行事綱)

### ① 反者道之動 — 先反後正

```
意念 → [反·外] 搜天下 → [反·內] 本地件 → [核] 從無到有
         (首選)         (次選)          (末選)
```

> **天下有 · 用天下的 · 天下近 · 改天下的 · 天下無 · 才自建.**

### ② 為而不爭 — 不重造輪子

- `萬法` 不改既有代碼, 僅 facade 聚合. 老代碼仍可直用.
- 新功能優先實現於最底層 (`dao_*`), 再由 `萬法` 暴露.
- 遇到多個實現 (如 SW live), 選最底層的 (memid 直連 > Builder 包裝).

### ③ 以神遇而不以目視 — 數理驗證
