### 3.2 带轮规格

- **主动带轮 (drive_pulley)**: 电机轴端, Φ120-140mm, 3-4槽 SPB
- **从动带轮 (driven_pulley)**: 主轴端, Φ400-500mm, 3-4槽 SPB
- 材质: HT200灰铁铸造
- 动平衡: G6.3级

### 3.3 传动比计算

```
电机转速: 1460 rpm
传动比 i = D从/D主 = 450/135 ≈ 3.33
主轴转速 n = 1460/3.33 ≈ 438 rpm
转子线速度 v = π×D×n/60 = π×0.4×438/60 ≈ 9.2 m/s
```

## 四、电气控制系统

### 4.1 起动方式

| 方式 | 适用功率 | 特点 |
|------|---------|------|
| **直接起动** | ≤11kW | 简单, 冲击电流大(7×In) |
| 星-三角起动 | 11-45kW | 起动电流降至1/3, 需6线电机 |
| **软起动器** | 任意 | 平滑起动, 减小冲击 |
| **变频器(VFD)** | 任意 | 调速+软启, 节能 |

### 4.2 推荐控制方案 (PC400×300, 11kW)

#### 方案A: 直接起动 (经济型)
- 断路器: DZ47-63 C32 (3P)
- 接触器: CJX2-2510 (25A)
- 热继电器: JR36-20 (14-22A)
- 总价: ~¥300-500

#### 方案B: 软起动器
- 型号: ABB PSR25-600-70 或 施耐德 ATS01N222
- 额定电流: 25A
- 起动时间: 5-15s 可调
- 总价: ~¥1500-3000

#### 方案C: 变频器 (推荐)
- 型号: ABB ACS310-03E-25A4 或 西门子 V20 6SL3210-5BE25-5UV0
- 额定功率: 11kW
- 额定电流: 25A
- 输入: 3相 380V 50Hz
- 输出: 0-380V, 0-50Hz
- 功能: 软启停, 调速, 过载保护, PID控制
- 总价: ~¥3000-5000

### 4.3 保护配置

| 保护功能 | 器件 | 整定值 |

## 五、关键零部件

### 5.1 筛板/篦条 (screen_plate)

- 材质: Mn13Cr2 高锰钢 或 铬锰合金
- 形式: 弧形篦条组, 包覆转子下半部
- 弧度: 约180° (半圆形)
- 篦条间距: 可调, 15-30mm (决定出料粒度)
- 篦条直径/宽度: 10-16mm
- 安装方式: 两端插入机壳侧壁凹槽, 可快速更换
- **设计要点**: 篦条弧面中心应与转子轴心同心, 篦条弧面内径 > 锤头回转半径 + 25mm间隙

### 5.2 轴承

- 型号: 6312 或 22312 (调心滚子轴承)
- 内径: Φ60mm (配合主轴)
- 外径: Φ130mm
- 轴承座: 剖分式 SN512, 铸铁
- 润滑: 锂基脂, 每3个月补脂

### 5.3 锤头

- 材质: 高铬铸铁 Cr26 或 高锰钢 Mn13
- 重量: 2-5kg/个 (PC400×300)
- 数量: 12-16个 (本模型: 16个, 4盘×4锤)
- 可调换工作面: 4个面
- 回转直径: Φ400mm

### 5.4 转子盘

- 材质: 45# 钢
- 厚度: 20-30mm
- 外径: ~Φ350mm
- 数量: 4片 (本模型)
- 键联接主轴, 销轴穿锤头

### 5.5 飞轮 (本模型未含)

- 安装位置: 主轴非传动端
- 作用: 储能, 平稳转速波动
- 重量: 约50-80kg (PC400×300)

## 六、机架与机壳

### 6.1 机壳 (casing)

- 材质: Q235B 焊接结构 或 HT200 铸铁
- 分体结构: 上壳 + 下壳 (水平分割)
- 壁厚: 15-25mm
- 进料口: 上壳顶部, 400×300mm
- 出料口: 下壳底部, 通过篦条筛分排出
- 侧板: 设有轴承座安装面

### 6.2 机架底座 (frame_base)

- 材质: Q235B 型钢焊接 或 铸铁
- 功能: 支撑整机, 安装电机底座
- 安装: 地脚螺栓 M16×4
