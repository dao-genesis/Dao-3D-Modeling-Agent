局部 bbox: `X[-40,40] Y[0,180] Z[0,40]`, 销孔 `(0, 120, 20)` 沿 Z 穿通

#### 原配方 vs 新配方

**原 PCD_DIRS (错)** — hammer 销孔未对齐销轴:

```python
PCD_DIRS = [
    ("T", +1,  0,    0,   0, -20),   # ty = +220 + 0 = 220
    ("F",  0, +1,  +90, +20,   0),   # tz = +220 + 0 = 220
    ("B", -1,  0, +180,   0, +20),
    ("K",  0, -1,  -90, -20,   0),
]
# T 方向锤头 ty=220, 局部 Y[0,180] → world Y[220,400]
# 销孔局部 Y=120 → world Y=340 ≠ 销轴 Y=220 (错开 120mm!)
# R_max = 400 > Ri=390 → 穿筛板 10mm
```

**新 PCD_DIRS (正)** — 销孔精确对齐销轴:

```python
PCD_DIRS_FIXED = [
    ("T", +1,  0,    0,  -120, -20),   # ty = +220 − 120 = 100
    ("F",  0, +1,  +90,  +20, -120),   # tz = +220 − 120 = 100
    ("B", -1,  0, +180, +120, +20),    # ty = −220 + 120 = −100
    ("K",  0, -1,  -90,  -20, +120),   # tz = −220 + 120 = −100
]
# T 方向锤头 ty=100, 局部 Y[0,180] → world Y[100, 280]
# 销孔局部 Y=120 → world Y=220 = 销轴 Y=220 ✓ 对齐
# R_max = 280 < Ri=390 → 间隙 110mm ✓ 不穿
```

#### 各方向旋转 + 销孔对齐数学验证

对每方向的 `Rx(ra)` 作用于销孔 `(0, 120, 20)`:

| 方向 | `Rx(ra)·(0,120,20)` | 目标销轴 world | ty | tz |
|---|---|---|---|---|
| T (ra=0) | (0, 120, 20) | (dx, 220, 0) | 220−120 = **100** | 0−20 = **−20** |
| F (ra=+90) | (0, −20, 120) | (dx, 0, 220) | 0+20 = **20** | 220−120 = **100** |
| B (ra=180) | (0, −120, −20) | (dx, −220, 0) | −220+120 = **−100** | 0+20 = **+20** |
| K (ra=−90) | (0, 20, −120) | (dx, 0, −220) | 0−20 = **−20** | −220+120 = **−100** |
